# React 学習記録

このリポジトリは、Reactの基礎からアプリ開発までの学習過程を記録するためのものです。  
最終的には Reactを使って小規模なWebアプリを自力で作れるレベル を目指します。

## 学習プラン

### Step1: JavaScript基礎（1〜3日）
- map / filter などの配列操作
- アロー関数
- 分割代入・スプレッド構文

### Step2: React基礎（4〜7日）
- Reactの基本概念
- JSX
- コンポーネント
- props
- 配列データのレンダリング（map）

### Step3: State管理（8〜14日）
- useState
- イベント処理
- 条件レンダリング
- フォーム入力管理

### Step4: ミニアプリ開発（15〜21日）
- カウンターアプリ
- TODOアプリ
- コンポーネント分割

### Step5: 実践アプリ（22〜30日）
- useEffect
- API取得（fetch）
- 簡単なWebアプリの作成

## 目標
- Reactの基本概念を理解する  
- 小規模なWebアプリを自力で作れるようになる  
- ポートフォリオとして公開できるアプリを開発する

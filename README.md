# React Study 🚀

React を使って **小規模な Web アプリを自力で開発できるようになること** を目的とした学習リポジトリです。

このリポジトリでは、JavaScript 基礎から始めて React を使ったアプリ開発まで段階的に学習していきます。

---

# 📚 学習目標

- React の基本概念を理解する
- React コンポーネント設計を理解する
- 小規模な Web アプリを開発できるようになる
- API を使ったアプリを作れるようになる
- ポートフォリオアプリを作成する

---

# 🛠 アプリ一覧

## 1. Counter App

React の基本となる **State 管理**を学ぶアプリ。

学習内容

- useState
- イベント処理
- コンポーネント

---

## 2. Todo App

タスク管理アプリ。

学習内容

- 配列 State 管理
- 追加 / 削除
- コンポーネント分割

---

## 3. Memo App

メモ管理アプリ。

学習内容

- CRUD の基本
- 状態管理
- UI 設計

---

## 4. GitHub User Search

GitHub ユーザー検索アプリ。

学習内容

- API 取得
- useEffect
- 非同期処理

---

## 5. Weather App

天気情報を取得するアプリ。

学習内容

- 外部 API
- ローディング管理
- エラーハンドリング

---

## 6. Portfolio App

最終的なポートフォリオアプリ。

作成予定アプリ例

- トレーニング記録アプリ
- 学習管理アプリ
- タスク管理アプリ
